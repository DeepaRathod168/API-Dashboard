## рџ”Њ API Endpoints

| Method | Route | Description |
|---|---|---|
| `GET` | `/api/targets` | List all monitored endpoints |
| `POST` | `/api/targets` | Register a new endpoint |
| `DELETE` | `/api/targets/:id` | Remove an endpoint |
| `GET` | `/api/logs?limit=N&url=X` | Raw ping logs (filterable) |
| `GET` | `/api/analytics` | Computed stats, timeline, endpoint details |
| `GET` | `/api/activity?limit=N` | Latest N ping events |
| `GET` | `/api/status` | Live status per target |
| `GET` | `/api/dummy/success` | Test endpoint (200 OK) |
| `GET` | `/api/dummy/slow` | Test endpoint (800ms delay) |
| `GET` | `/api/dummy/error` | Test endpoint (500 Error) |
